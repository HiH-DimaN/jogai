## 1. ОБЗОР ПРОДУКТА

### 1.1. Суть

Jogai — мультиканальная AI-платформа для гемблинг-аудитории LATAM. Помогает игрокам находить выгодные бонусы, анализировать условия казино. Монетизация: CPA + RevShare. Мультиязычность заложена с первого дня.

### 1.2. Проблема

Тысячи каналов постят «Бонус 500%!!!» без анализа. Игроки не понимают вейджеры и реальные шансы. Нет инструмента для математической оценки.

### 1.3. Решение

AI за 5 секунд: «Этот бонус — ловушка. Вейджер x60 = R$180 000. Вот три бонуса, которые РЕАЛЬНО выгодны.» На родном языке пользователя.

### 1.4. Целевая аудитория

- **Фаза 1:** Бразилец, 22–40 лет, мужчина, играет в онлайн-казино, сидит в Telegram
- **Фаза 3:** Мексиканец с аналогичным профилем
- **Фаза 4:** Эквадор, Чили, Перу

## 3. МУЛЬТИЯЗЫЧНОСТЬ

### 3.1. Принцип

Весь пользовательский контент на языке пользователя. Код один — тексты в JSON-файлах локализации. Добавление нового языка = перевод JSON, ноль изменений кода.

### 3.2. Что определяет язык

1. Сохранённый locale пользователя (приоритет)
2. Telegram language_code (pt → PT-BR, es → ES-MX)
3. Фоллбэк: PT-BR

### 3.3. Что переводится

| Элемент | Где хранится перевод |
|---------|---------------------|
| Все тексты бота (кнопки, ответы, ошибки) | `locales/pt_BR.json` |
| Интерфейс Mini App | `miniapp/public/locales/pt-BR/translation.json` |
| Интерфейс лендинга | `landing/messages/pt-BR.json` |
| Названия бонусов | БД: `title_pt`, `title_es` |
| Описания казино | БД: `description_pt`, `description_es` |
| Ответы AI | AI отвечает на `{language}` из промпта |
| Вердикты Jogai Score | Через ключ i18n: `t("verdict_excellent", locale)` |
| Форматирование (валюта, даты) | `formatters.py` по locale |

### 3.4. Что НЕ переводится

- Код и комментарии (английский)
- Внутренняя документация (русский)
- Техническая аналитика (английский)

## 4. ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ

### 4.1. Telegram-бот (Фаза 1)

#### F1.1 Welcome Flow
- /start → приветствие на языке пользователя: `t("welcome", locale)`
- 4 inline-кнопки с текстом из locales
- Deep link `/start ref_CODE` → реферал
- Сохранение user с locale и geo

#### F1.2 /bonus — Лучшие бонусы
- Топ-3 бонуса WHERE geo содержит geo пользователя
- Текст: `title_pt` или `title_es` по locale
- Вердикт: `t(verdict_key, locale)` → «BOM NEGÓCIO» или «BUEN NEGOCIO»
- Валюта: `format_currency(amount, locale)` → «R$1.500» или «MX$1,500»
- Кнопка `t("btn_get_bonus", locale)` → реферальная ссылка

#### F1.3 /analyze — AI-анализ бонуса
- Промпт: `t("analyze_prompt", locale)`
- AI парсит на любом языке, отвечает на `{language}` пользователя
- Результат: все строки через `t()` с подстановкой чисел
- Валюта: `format_currency()` по locale

#### F1.4 /casino — AI-подбор
- 5 вопросов: все тексты и варианты ответов из locales
- Суммы депозитов в валюте ГЕО (R$ для BR, MX$ для MX)
- Результат: казино отфильтрованы по geo пользователя

#### F1.5 /sport — Ставка дня
- Фильтр по geo: Série A для BR, Liga MX для MX
- Текст: `pick_description_pt` или `_es` по locale

#### F1.6 Реферальная система
- Все тексты через `t()`: `t("referral_your_link", locale)`
- Jogai Coins — одинаковая механика для всех ГЕО

### 4.2. Telegram-канал (Фаза 1)

#### F2.1 Расписание (на языке канала)
- 09:00 — Бонус дня: `t("channel_bonus_day", channel_locale, ...)`
- 14:00 — Обзор слота: `t("channel_slot_review", channel_locale, ...)`
- 18:00 — Ставка дня (Série A / Liga MX по ГЕО)
- 21:00 — Занос подписчика

#### F2.2 Мульти-канал (Фаза 3)
- @jogai_br → контент на PT-BR, бонусы geo=BR, футбол Série A
- @jogai_mx → контент на ES-MX, бонусы geo=MX, футбол Liga MX
- Одна Celery задача, два канала, два языка

### 4.3. SEO-лендинг (Фаза 1)

#### F3.1 Мультиязычные URL
- jogai.fun/pt-BR/ → главная на PT-BR (`<html lang="pt-BR">`)
- jogai.fun/es-MX/ → главная на ES-MX (фаза 3)
- jogai.fun/ → редирект на pt-BR (дефолт)

#### F3.2 Контент по locale
- Таблица казино: `description_pt` / `description_es`
- Таблица бонусов: `title_pt` / `title_es`, валюта по locale
- Meta tags, OG tags — на языке страницы
- JSON-LD — локализованный
- `<link rel="alternate" hreflang="pt-BR" />` для SEO

### 4.4. Mini App (Фаза 2)

#### F4.1 Автоопределение языка
- Из Telegram WebApp `language_code` → `i18n.changeLanguage()`
- Фоллбэк: PT-BR
- Все компоненты: `{t('key')}`

#### F4.2 Контент по locale
- Дашборд: бонусы geo пользователя, текст на locale
- Анализатор: AI отвечает на языке пользователя
- Квиз: вопросы и варианты из locales
- Валюта: R$ / MX$ по locale

### 4.5. Трекер ставок (Фаза 3)

- Валюта записи: BRL или MXN (по geo пользователя)
- Интерфейс: на locale пользователя
- Статистика: в валюте пользователя

### 4.6. PRO-подписка (Фаза 3)

- Цена: R$24,90/мес (BR), MX$99/мес (MX)
- Описание: на locale пользователя
